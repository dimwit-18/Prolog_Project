******************************************************************************************

# HOW TO RUN THE PROGRAM :-

To run the program, follow either of the procedures 

    method 1:
    1. open terminal.
    2. open SWI-Prolog by invoking swipl.
    3. type [queens]. to load the program.
    
    method 2;
    1. open terminal and naviage to the directory where the program is present.
    2. enter swipl -s queens.pl to load the program.
    
# INPUT DETAILS :-

 Follow the following procedure to enter input and get desired output.
 
 1. To perform DFS, simply type 'solveDFS.' (without quotes) and press enter.
 2. To perform BFS, simply type 'solveBFS.' (without quotes) and press enter.
 3. Output is displayed.
 
# OUTPUT DETAILS :-
 
 1. The output in case of DFS is all the possible chessboard combinations (92 in total),
    each of form :
    [1/Q1, 2/Q2,......8/Q8].
 2. The output in case of BFS is the list of all possible chessboard combinations 
    (92 in total), of form :
   [[1/Q11, 2/Q12,......8/Q18], [1/Q21, 2/Q22,......8/Q28],......[1/Q921, 2/Q922,......8/Q928]]
